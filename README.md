## 🇬🇧 English

### What is MenuMaster2?

MenuMaster2 teaches the macOS Finder everything Apple left out: right-click actions for paths, hashes, PDF compression, email sending and your own document templates. Plus work modes that launch and close your apps at the press of a button, window shortcuts like "minimize all" or "maximize", a Windows-style traffic light to truly close windows instead of going fullscreen, a keep-awake mode and a cleaning mode that locks your keyboard and trackpad.

Built for people who live in Finder and want it to work *for* them.

### Features

#### 🗂 Finder Context Menu (right-click in Finder)

- **Copy path** — absolute, relative, as URL, as Markdown link, or POSIX quoted
- **Calculate hashes** — MD5, SHA-1, SHA-256 for any file
- **Compress PDFs** — optimize file size with adjustable quality
- **Send via email** — attach selection directly to a new Mail/Outlook message
- **Open in app** — launch folder in Terminal, iTerm, VS Code, or any other app
- **Rename with presets** — define your own naming patterns (e.g. "Invoice", "Scan")
- **Batch rename** — patterns with date, counter, and original name placeholders
- **Path variants** — quick copy in five different formats

#### 📄 New Document (from templates)

- **Built-in templates** — Word, Excel, Pages, Numbers, Keynote, PDF, text files
- **User templates** — drop any file into the templates folder, it becomes available instantly
- **Rename on create** — name the new file before it lands on your Desktop

#### 💼 Work Modes

- **Launch multiple apps** at once when you start a work session
- **Close apps** when you're done — back to a clean desktop in one click
- **Custom app sets** — one for focus work, one for meetings, one for weekends, whatever you need

#### 🪟 Window Shortcuts

- **Minimize all windows** — global shortcut
- **Maximize window** — true maximize, not fullscreen
- **Windows-style traffic light** — the red close button actually closes the window, no more fullscreen-by-accident

#### ⚡ System Modes

- **Keep Awake** — prevent your Mac from sleeping when you need it to stay on (downloads, long renders, presentations)
- **Cleaning Mode** — lock keyboard and trackpad to wipe your screen and clean the surface

### Installation

1. Download the latest `.dmg` from [Releases](../../releases)
2. Drag MenuMaster2 to your Applications folder
3. On first launch, grant the permissions MenuMaster2 asks for:
   - **Finder Extension** — enable in System Settings → Privacy & Security → Extensions → Finder
   - **Accessibility** — for window shortcuts
   - **Automation** — to launch other apps in work modes

That's it. The Finder context menu is active immediately.

### Requirements

- macOS 13 (Ventura) or newer
- Apple Silicon or Intel

## 🇩🇪 Deutsch

### Was ist MenuMaster2?

MenuMaster2 bringt dem macOS-Finder alles bei, was Apple weggelassen hat: Rechtsklick-Aktionen für Pfade, Hashes, PDF-Kompression, E-Mail-Versand und eigene Dokumentvorlagen. Dazu Arbeitsmodi, die deine Apps per Knopfdruck starten und schließen, Fenster-Shortcuts wie „alle minimieren" oder „maximieren", eine Windows-ähnliche Ampel zum echten Schließen statt Vollbild, Wachhalte-Modus und ein Reinigungsmodus, der Tastatur und Trackpad sperrt.

Gemacht für Leute, die im Finder leben und wollen, dass er *für* sie arbeitet.

### Funktionen

#### 🗂 Finder-Kontextmenü (Rechtsklick im Finder)

- **Pfad kopieren** — absolut, relativ, als URL, als Markdown-Link oder POSIX-quoted
- **Hashes berechnen** — MD5, SHA-1, SHA-256 für jede Datei
- **PDFs komprimieren** — Dateigröße optimieren mit einstellbarer Qualität
- **Per E-Mail senden** — Auswahl direkt an eine neue Mail/Outlook-Nachricht anhängen
- **Öffnen in App** — Ordner in Terminal, iTerm, VS Code oder beliebiger anderer App öffnen
- **Umbenennen mit Presets** — eigene Namensmuster definieren (z. B. „Rechnung", „Scan")
- **Batch-Umbenennen** — Muster mit Datum, Zähler und Originalname als Platzhalter
- **Pfad-Varianten** — schnelles Kopieren in fünf verschiedenen Formaten

#### 📄 Neues Dokument (aus Vorlagen)

- **Integrierte Vorlagen** — Word, Excel, Pages, Numbers, Keynote, PDF, Textdateien
- **Eigene Vorlagen** — leg eine beliebige Datei in den Vorlagenordner, sie ist sofort verfügbar
- **Beim Erstellen umbenennen** — Dateinamen festlegen, bevor die Datei auf dem Schreibtisch landet

#### 💼 Arbeitsmodi

- **Mehrere Apps gleichzeitig starten**, wenn du eine Arbeitssession beginnst
- **Apps schließen**, wenn du fertig bist — zurück zum aufgeräumten Desktop mit einem Klick
- **Eigene App-Sets** — einer für fokussierte Arbeit, einer für Meetings, einer fürs Wochenende

#### 🪟 Fenster-Shortcuts

- **Alle Fenster minimieren** — globales Tastaturkürzel
- **Fenster maximieren** — echtes Maximieren, kein Vollbild
- **Windows-artige Ampel** — die rote Schließen-Taste schließt wirklich das Fenster, kein versehentlicher Vollbildmodus mehr

#### ⚡ System-Modi

- **Wachhalten** — verhindert, dass dein Mac schläft, wenn er wach bleiben soll (Downloads, lange Renders, Präsentationen)
- **Reinigungsmodus** — Tastatur und Trackpad sperren, um den Bildschirm zu wischen und die Oberfläche zu reinigen

### Installation

1. Lade die aktuelle `.dmg` aus [Releases](../../releases)
2. Ziehe MenuMaster2 in deinen Programme-Ordner
3. Beim ersten Start erteile die Berechtigungen, nach denen MenuMaster2 fragt:
   - **Finder-Erweiterung** — aktivieren unter Systemeinstellungen → Datenschutz & Sicherheit → Erweiterungen → Finder
   - **Bedienungshilfen** — für Fenster-Shortcuts
   - **Automation** — um in Arbeitsmodi andere Apps zu starten

Das war's. Das Finder-Kontextmenü ist sofort aktiv.
